# 🚀 React proyect Gamispace

This repository contains a react web application structured into:

- `client/`: Frontend application built with React.
- `server/`: Backend API built with Node.js and Express.
- `gamispace.sql`: SQL script to set up the database schema.

## 📦 Requirements

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)
- [MySQL](https://www.mysql.com/) or a compatible SQL database engine

## 🛠️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### 2. Configure the Backend

```bash
cd server
npm install
```

### 3. Start the Backend

```bash
node server.js
```

### 4. Configure the Frontend

```bash
cd ../client
npm install
```

### 5. Start the Frontend

```bash
npm run dev
```

## 🛠️ StoryBook Instructions

### 1. Run StoryBook

